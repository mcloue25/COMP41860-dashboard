## Information Point:

The  following  document  captures  regulatory  amendments.  It  does  not  include  minor  editorial amendments which do not affect the intent of the Regulation in question.

The 'Affected Section' refers to the numbering as per the version listed in 'Version Amended'. In other words, all numbers listed below refer to the version to which changes have been made.
